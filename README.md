## Сборка
Для компиляции:
```bash
make
```
## Синтаксис
Переменные создаются через оператор "=". 
Каждая строка должна заканчиваться ";" 
Поддерживаются +, -, *, /, (), <, >, ==

## Функции
Функция print() выводит данные на экран.
Есть функции while, for, if

## Использование
Создайте файл .chago и запустите его:
```bash
./chago <file_name>.chago
```
